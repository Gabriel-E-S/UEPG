Bem-vindo ao meu repositório central de estudos!

Este espaço é dedicado a documentar minha jornada de aprendizado na área de tecnologia e computação. Aqui eu guardo meus códigos, exercícios práticos, trabalhos acadêmicos e projetos desenvolvidos ao longo dos meus estudos.

## 📂 Estrutura do Repositório

O repositório está organizado por disciplinas/temas para facilitar a navegação. Abaixo está a descrição do que você pode encontrar em cada pasta:

* 📁 **`Análise de Algoritmos/`**: Estudos sobre complexidade de algoritmos (Big O), ordenação e otimização de código.
* 📁 **`Banco de Dados/`**: Modelagem de dados, scripts SQL, consultas e estruturação de bancos relacionais.
* 📁 **`Computação Gráfica/`**: Conceitos de renderização, manipulação de imagens e algoritmos gráficos.
* 📁 **`Desenvolvimento Web/`**: Projetos e exercícios focados em Front-end e Back-end (HTML, CSS, JS, etc.).
* 📁 **`Estrutura de Dados/` e `Estrutura de Dados II/`**: Implementação de listas, pilhas, filas, árvores, grafos e algoritmos de busca.
* 📁 **`Organização e Arquitetura de Computadores/`**: Entendimento de hardware, assembly, portas lógicas e funcionamento interno da máquina.
* 📁 **`Programação Orientada a Objetos/`**: Exercícios aplicando os pilares da POO (Classes, Herança, Polimorfismo, Encapsulamento).
* 📁 **`Extras/`**: Cursos paralelos, desafios de lógica, testes e experimentações soltas.

*(Sinta-se à vontade para explorar as pastas! Alguns projetos específicos podem conter seus próprios arquivos README detalhando a implementação).*

## 🛠️ Como explorar este repositório

Se desejar baixar os códigos para testar localmente:

1. Clone o repositório:
   ```bash
   git clone https://github.com/Gabriel-E-S/UEPG.git
   ```

2. Navegue até a pasta da disciplina desejada:

3. Execute o código conforme a linguagem utilizada na pasta (ex: gcc para C, javac para Java, etc...).
