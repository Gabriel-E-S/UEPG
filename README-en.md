Welcome to my central study repository! 🚀

This space is dedicated to documenting my learning journey in technology and computer Engineering. Here I store my code, practical exercises, academic assignments, and projects developed throughout my studies.

## 📂 Repository Structure

The repository is organized by subjects/themes to make navigation easier. Below is a description of what you can find in each folder:

* 📁 **`Análise de Algoritmos/` (Algorithm Analysis)**: Studies on algorithm complexity (Big O notation), sorting, and code optimization.
* 📁 **`Banco de Dados/` (Databases)**: Data modeling, SQL scripts, queries, and relational database structuring.
* 📁 **`Computação Gráfica/` (Computer Graphics)**: Rendering concepts, image manipulation, and graphic algorithms.
* 📁 **`Desenvolvimento Web/` (Web Development)**: Projects and exercises focused on Front-end and Back-end (HTML, CSS, JS, etc.).
* 📁 **`Estrutura de Dados/` & `Estrutura de Dados II/` (Data Structures I & II)**: Implementation of lists, stacks, queues, trees, graphs, and search algorithms.
* 📁 **`Organização e Arquitetura de Computadores/` (Computer Organization & Architecture)**: Understanding of hardware, assembly, logic gates, and the internal workings of machines.
* 📁 **`Programação Orientada a Objetos/` (Object-Oriented Programming)**: Exercises applying OOP pillars (Classes, Inheritance, Polymorphism, Encapsulation).
* 📁 **`Extras/`**: Parallel courses, logic challenges, tests, and random experiments.

*(Feel free to explore the folders! Some specific projects might contain their own README files detailing the implementation).*
